■ 汎用スケジュール取り込み サンプルCSV

このフォルダにサンプルCSVが含まれています。
設定画面「スケジュール取り込み」タブで取り込み設定を作成し、
このフォルダを監視フォルダに指定してお試しください。

【サンプルファイル一覧】
  sample_surgery.csv     … 手術スケジュール（日付+時刻が別列）
  sample_exam.csv        … 検査スケジュール（日時が1列）
  sample_transport.csv   … 外部移送スケジュール（所要時間列あり）

【カラムマッピング設定の例】

■ sample_surgery.csv
  日付列   : 手術日
  時刻列   : 開始時刻
  タイトル列: 術式
  ID列     : 患者ID
  所要時間列: 予定時間(分)

■ sample_exam.csv
  日付列   : 検査日時 ← 日時が1列に入っている場合はここに入力し時刻列は空欄
  タイトル列: 検査名
  ID列     : 患者番号

■ sample_transport.csv
  日付列   : 日付
  時刻列   : 出発時刻
  タイトル列: 行き先・目的
  ID列     : 患者ID
  所要時間列: 所要時間(分)
